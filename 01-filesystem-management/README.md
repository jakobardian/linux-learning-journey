# Module 01 - Filesystem Management
Hands-on practice of core Linux filesystem operations 
with structured execution and validation.

## Overview
This module covers core Linux filesystem operations from file organization and search
to archive management and link types executed and validated through the terminal.

## Learning Objective
By the end of this module,  the following skills will be achieved:

- Navigating and organizing the Linux filesystem using CLI
- Searching files efficiently by name, type, size, and modification time
- Creating, compressing, and extracting archives with tar and gzip
- Understanding between hard links and symbolic links

## Scope
- Directory and file organization
- File search and analysis
- Archive and compression
- Hard and symbolic links

## Lab Environment
- Virtual Machine: VMware Workstation Pro 25H2
- Operating System: Ubuntu 24.04 LTS
- Shell: Zsh
