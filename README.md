# Linux Learning Journey
This repository contains my hands-on practice in Linux system administration.

## Learning Objective
This project focuses on building strong Linux system administration 
fundamentals through structured hands-on practice and simulated 
scenarios in a virtual machine environment.

## Learning Module
The lab is structured into multiple modules covering:

- User and access management
- Service management (systemd)
- Process monitoring
- Log troubleshooting
- Networking basics
- Storage and disk handling
- Security configuration
- Multi-VM practice setup
- Bash automation
- Incident simulation & basic RCA

## Lab Environment
- OS: Ubuntu Desktop ver.24.04 LTS
- Virtual Machine: VMware Workstation Pro ver.25H2
